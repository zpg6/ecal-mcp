# Scenario 03: topic_typo

A user reports: **"My subscriber on `/sensor/lidar` isn't getting any
messages. The lidar driver is supposed to be running."**

Diagnose. Your final answer should:

- State exactly what's wrong with the user's setup
- Recommend the precise topic they should be subscribing to instead
- (Optional) call out anything else they should know about the lidar
  topology on this network

## Capture as you go

After each tool call, briefly note:

- Which tool you reached for and **why** (your reasoning, not just the name)
- Any moment where the description, schema, or output shape made you guess
- Anything you wish the tool returned but didn't, or did return but cluttered things
