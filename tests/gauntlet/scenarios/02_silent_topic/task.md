# Scenario 02: silent_topic

The robot is unresponsive. Operator says: **"`/cmd_vel` looks dead — the
robot isn't moving."**

Diagnose what's wrong. Be specific in your final answer:

- What is and isn't registered on `/cmd_vel`?
- What's the root cause (in one sentence)?
- What concrete remediation should the operator take?

## Capture as you go

After each tool call, briefly note:

- Which tool you reached for and **why** (your reasoning, not just the name)
- Any moment where the description, schema, or output shape made you guess
- Anything you wish the tool returned but didn't, or did return but cluttered things
