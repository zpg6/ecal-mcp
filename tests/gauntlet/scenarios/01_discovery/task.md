# Scenario 01: discovery

You are dropped into a running robot system. Nothing is broken, but you've
never seen this stack before. **Survey the network and produce a concise
system summary**:

- What hosts are participating?
- What topics flow on the network, and who publishes / subscribes to each?
- What services exist, on which host, and with what methods?
- Any observations (e.g. publishers without subscribers, mixed-role hosts)?

Use only the eCAL MCP tools (via the CLI helper described in your
instructions). End with a short structured summary suitable for a teammate
who has never seen this system.

## Capture as you go

After each tool call, briefly note (in your reply / scratchpad):

- Which tool you reached for and **why** (your reasoning, not just the name)
- Any moment where the description, schema, or output shape made you guess
- Anything you wish the tool returned but didn't, or did return but cluttered things
